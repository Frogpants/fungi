# README

### Folder Hierarchy
```
Game/
├── Assets/
│   ├── Models/
│   ├── Textures/
│   ├── Shaders/   (optional overrides)
│   └── Prefabs/
|
├── Scenes/
│   └── Game.scene
|
├── Scripts/
│   └── Game.cpp
│
└── Main.cpp
```

### Setup Commands

Setup for project
```bash
cd build
cmake ..
```

Update project code
```bash
make
```

Run project
```bash
./MyEngine
```
